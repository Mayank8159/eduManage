# Frontend - EduManage Dashboard

## Scripts

- npm run dev
- npm run build
- npm start

## Environment Variables

Create .env.local from .env.example:

- NEXT_PUBLIC_API_URL=http://localhost:5000/api/v1

## Role Dashboards

- Principal: analytics, logs, reports, teacher directory
- Teacher: class view, attendance/marks upload, AI simulation
- Student: marks/attendance charts, feedback, notifications

## UI Stack

- Next.js App Router
- Tailwind CSS
- Recharts
- Axios
